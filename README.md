# transport
Transport app for real time tracking of personal vehicles using realtime Database and Realtime GPS 

Using API and Platforms 

Firebase DataBase 
Firebase Auth
Firebase Cloud Notification 
Firebase Analytics 
Google Maps API
Digital Ocean for Server
Node.js for API 
Mango DB for API Storage
